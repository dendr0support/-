# ⌨️ Слепая печать — тренажёр

Тренажёр для обучения слепой печати. Настройка символов, таймер обратного отсчёта, живая статистика.

* * *

### 🧠 Для кого это

💻 Для всех, кто хочет печатать быстрее

* * *

### ✨ Возможности

- Русская и английская раскладка
- Выбор символов: буквы, цифры, знаки
- Режим слов (пробел генерирует слова 1–10 букв)
- Настраиваемый Таймер обратного отсчёта
- Статистика: CPM, точность, ошибки, время
- Визуальная подсветка текущего символа и ошибок
- Управление с клавиатуры

* * *

### 🎮 Как пользоваться

1. **Настройки** — выбери раскладку, символы, при необходимости таймер
2. **Тренировка** — печатай подсвеченные символы
3. **Результаты** — смотри статистику, начинай новую тренировку

* * *

### ⌨️ Горячие клавиши

| Меню | Клавиши | Действие |
|-------|---------|----------|
| Любое | `Enter` / `Backspace` / `Esc` | Перейти в следующее меню |
| Настройки | `любая клавиша` | Включить/выключить символ |

* * *
